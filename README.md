# Homebrew Tap

Personal Homebrew tap for my tools.

## Install

```bash
brew tap MichaelVessia/tap
brew install <formula>
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [fmcal](https://github.com/MichaelVessia/fmcal) | CLI for Fastmail calendars via CalDAV |
| [paperless-cli](https://github.com/MichaelVessia/paperless-cli) | CLI for Paperless-ngx document management |
| [x-to-obsidian](https://github.com/MichaelVessia/x-to-obsidian) | Scrape X bookmarks to Obsidian vault |
